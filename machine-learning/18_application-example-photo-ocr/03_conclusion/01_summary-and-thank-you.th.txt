ยินดีต้อนรับสู่วิดีโอสุดท้ายของวิชา Machine Learning เราผ่านกันมาหลากหลายบทเรียนแล้ว วิดีโอนี้ ผมขอสรุปหัวข้อหลัก ของวิชานี้อย่างรวดเร็ว แล้วกล่าวอีกเล็กน้อยในตอนจบ และนั่นก็เป็นการสรุปสำหรับวิชานี้ เอาล่ะ สิ่งที่เราได้ทำจบนั้นมีอะไรบ้าง บทเรียนนี้ เราได้ใช้เวลาไปมากกับ supervised learning algorithms อย่างเช่น linear regression, logistic regression, neural networks, SVMs ปัญหาที่ข้อมูลและตัวอย่างมีการระบุชื่อ(labelled) อย่างเช่น x(i), y(i) และเราก็ใช้เวลาไม่น้อยไปกับ unsupervised learning อย่างเช่นเรื่อง K-means clustering, Principal Components Analysis ใน dimensionality reduction และ Anomaly Detection algorithms เมื่อมีแต่ ข้อมูลที่ไม่ระบุชื่อ แม้ว่า Anomaly Detection ยังใช้ได้กับข้อมูลที่ระบุชื่อ เพื่อประเมิน algorithm เรายังมีเวลาได้พูดเกี่ยวกับแอพพลิเคชั่นพิเศษ หรือหัวข้อพิเศษอย่างเช่น Recommender Systems และ ระบบ machine learning ขนาดใหญ่ รวมทั้งระบบ parallelized และ map-reduce และแอพพลิเคชั่นพิเศษอื่น อย่างเช่น sliding windows ซึ่งใช้กับ classification วัตถุ ใน computer vision และยังใช้เวลากับหลายเรื่องที่เกี่ยวกับ คำปรึกษาในการสร้างระบบ machine learning ทั้งเกี่ยวกับการทำความเข้าใจว่า อะไรที่ทำให้ machine learning algorithm ใช้ได้หรือไม่ได้ จึงได้พูดเกี่ยวกับ bias และ variance และการใช้ regularization แก้ปัญหาของ variance เรายังได้พูดอีกเล็กน้อยว่า เราจะตัดสินใจอย่างไรว่า เราจะต้องทำอะไรต่อ ดังนั้น การจัดลำดับก่อนหลังว่า เราจะใช้เวล่าอย่างไร ในขณะที่กำลังพัฒนาระบบ machine learning ได้พูดเกี่ยวกับการประเมินผล learning algorithm เมตริกซ์ที่ใช้ประเมิน เช่น precision recall, F1 score และการประเมินผลในภาคปฏิบัติ เช่น trainingt set, cross-validation se และ test set และเราก็ได้ใช้เวลาไปมากในการพูดถึง การแก้ไขจุดบกพร่องใน learning algorithm และ ให้แน่ใจว่า learning algrorithm ใช้งานได้ และเรายังพูดถึงการวินิจฉัยต่างๆ เช่น learning curve และพูดเรื่องเกี่ยวกับ error analysis และ ceiling analysis เครื่องมือเหล่านี้จะช่วยคุณตัดสินใจ ว่าจะทำอะไรต่อ และควรใช้เวลาอันมีค่าของคุณอย่างไร ในขณะที่กำลังพัฒนาระบบ machine learning นอกเหนือจากการมีเครื่องมือ machine learning พร้อมใช้งานแล้ว การได้รู้จักเครื่องมือต่างๆ ของ machine learning เช่น supervised learning และ unsupervised learning และอื่นๆ ผมหวังว่า ไม่เพียงมีเครื่องมือเท่านั้น แต่คุณสามารถประยุกต์ใช้เครื่องมือเหล่านี้ได้เป็นอย่างดี ในการสร้างระบบ machine learning ที่มีประสิทธิภาพ เอาล่ะ จบแล้วครับ นั่นก็เป็นหัวข้อต่างๆ ในบทเรียนนี้ และเมื่อคุณได้ทำทุกอย่างตามบทเรียนนี้โดยตลอด คุณก็สามารถบอกกับตัวเองได้ว่าคุณคือ ผู้เชี่ยวชาญ machine learning อย่างที่รู้กัน machine learning เป็นเทคโนโลยี ที่มีผลกระทบมากในทางวิทยาศาสตร์ เทคโนโลยี และอุตสาหกรรม และบัดนี้คุณเป็นผู้มีคุณสมบัติเหมาะสมที่จะใช้เครื่องมือ machine learning ที่จะก่อให้เกิดผลอันยิ่งใหญ่ได้ ผมหวังว่าพวกคุณในชั้นเรียนนี้ จะพบหนทางในการนำ machine learning ไปใช้ ในการสร้างระบบและแอพพลิเคชั่นที่ยอดเยี่ยม และผลิตภัณฑ์ยอดเยี่ยมด้วย และหวังว่าพวกคุณจะพบหนทาง ในการนำ machine learning ไปใช้ไม่เพียง ทำให้ชีวิต<i>ของคุณ</i>ดีขึ้น แต่ยังทำให้ชีวิตผู้อื่นอีกมากมายดีขึ้นด้วย ผมอยากบอกว่า ผมสนุกในการสอนบทเรียนนี้เป็นอย่างมาก ขอบคุณด้วยครับ ก่อนสรุป สิ่งสุดท้ายที่ผมอยากจะบอก นั่นคือมันอาจไม่นานนัก ที่ผมเคยเป็นนักศึกษา และแม้แต่วันนี้ผมก็ยังคงศึกษาเพิ่มเติมเมื่อมีเวลา เพื่อจะได้เรียนรู้สิ่งใหม่ และผมก็รู้ว่ามันใช้เวลามาก ที่จะเรียนรู้สิ่งต่างๆ ผมรู้ว่าเป็นไปได้ที่คุณจะมีงานยุ่ง มีสิ่งต่างๆเกิดขึ้นมากมายในชีวิต และความเป็นจริงคือคุณก็ยังมี เวลาหรือหาเวลาเพื่อดูวิดีโอนี้ และวิดีโอที่มากมายเหล่านี้ ใช้เวลาหลายชั่วโมงทีเดียวใช่ไหมครับ และความเป็นจริงคือคุณก็ใช้เวลา เพื่อศึกษาส่วนที่เป็นคำถามทบทวน และพวกคุณไม่น้อยที่ใช้เวลา ทำแบบฝึกหัดโปรแกรมมิ่ง และนั่นก็เป็นแบบฝึกหัดที่ทั้งยาวและยาก ผมอยากจะบอกว่าขอบคุณมากครับ และผมรู้ว่าคุณได้ศึกษาอย่างหนักในบทเรียนนี้ และคุณก็ใช้เวลาไปมากกับบทเรียนนี้ พวกคุณไม่น้อยได้ทุ่มเทไปมากกับบทเรียนนี้ ดังนั้นผมหวังว่าคุณจะได้รับประโยชน์จากบทเรียนนี้ไปมากเช่นกัน และผมอยากบอกว่า ขอบคุณมากครับที่ได้เป็นนักศึกษาในบทเรียนนี้
